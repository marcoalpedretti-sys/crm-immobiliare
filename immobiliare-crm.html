Configura l'integrazione con Google Calendar

Google Calendar API

Per connettere Google Calendar segui questi passaggi:

1. Vai su console.cloud.google.com
2. Crea un progetto → abilita Google Calendar API
3. Crea credenziali OAuth 2.0 → Web application
4. In Authorized JavaScript origins inserisci:
  https://marcoalpedretti-sys.github.io
5. In Authorized redirect URIs inserisci:
  https://marcoalpedretti-sys.github.io/crm-immobiliare/immobiliare-crm.html
6. Salva e aspetta 1-2 minuti
7. Incolla il Client ID qui sotto e salva
