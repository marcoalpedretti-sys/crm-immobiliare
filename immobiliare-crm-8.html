Configura l'integrazione con Google Calendar

Google Calendar API

Per connettere Google Calendar segui questi passaggi:

1. Vai su console.cloud.google.com
2. Crea un progetto → abilita Google Calendar API
3. Crea credenziali OAuth 2.0 → Web application
4. In Authorized JavaScript origins: lascia vuoto
5. In Authorized redirect URIs: inserisci il percorso completo del file, es:
  • Windows: file:///C:/Users/TuoNome/Downloads/immobiliare-crm.html
  • Android (Chrome): apri il file, copia l'URL dalla barra indirizzi e incollalo qui
6. Salva e aspetta 1 minuto
7. Incolla il Client ID qui sotto e salva

Verrà pre-selezionato automaticamente al momento della connessione
